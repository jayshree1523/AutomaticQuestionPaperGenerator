# ABSTRACT:-
In this project, I have tried to build an Automatic Question Paper Generator website, which can simulate all the basic
controls that an actual required for generating the paper.


I have used Netbeans14 , java 1.8, tomcat 9.0 and MySQL 8.0.28.


# FUNCTIONALITIES:-
1.The teacher should login into the system with unique her/his username and password.
2.If the user name and password are valid then teacher can gain access to the system.
3.Teacher can Add New Question.
4.Teacher can See All Questions.
5.Teacher can Generate Paper.
6.Teacher can Print the Paper.


# INTRODUCTION:-
“ONLINE QUESTION PAPER GENERATOR ” is a website which is used to create exam papers quickly.It has been developed to
 reduce the manual work of teachers and is reduced as much as possible to avoid errors while entering data manually.
